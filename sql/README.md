# SQL Files

This folder contains the SQL scripts used to create and analyze the Delta Hiring Database.

## schema.sql
Defines the relational database structure including:
- CREATE TABLE statements
- Primary keys
- Foreign key constraints
- Data types and relationships

## queries.sql
Contains sample SELECT queries demonstrating:
- INNER JOIN operations
- Multi-table relationships
- Aggregate functions (COUNT, GROUP BY)
- Data retrieval for reporting and analysis

All SQL syntax is written for Microsoft Access.
