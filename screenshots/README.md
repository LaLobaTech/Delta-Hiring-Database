# Query Result Screenshots

This folder contains sample outputs from the SQL queries executed in Microsoft Access. These screenshots demonstrate the relational joins and aggregate functions used in the Delta Hiring Database.

## 1. Candidate Applications Query
Displays candidate names, job titles, and application statuses using INNER JOIN across multiple tables.

## 2. Department Application Count
Shows the total number of applications received by each department using JOIN and GROUP BY.

## 3. Candidate Assessment Results
Displays assessment type, score, and date for each candidate using relational joins.
